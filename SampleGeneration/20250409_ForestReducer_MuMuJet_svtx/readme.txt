TO SKIM WITH SVTX OPTION SIMPLY USE THE FLAG: 

" --svtx true "

SVTX SKIMMER CREATES A LIST OF TRACKS AND SVTX FOR EACH JET AS 2D VECTORS. 
COMPILING THESE REQUIRES AN ADDITIONAL DICTIONARY.
TO READ THE FILES POST-SKIM THE FOLLOWING LINES WILL BE EXECUTED IN YOUR CLEAN.SH 

rootcling -f DictDict.cxx -c DictTypes.h LinkDef.h
g++ -fPIC -shared DictDict.cxx `root-config --cflags --libs` -o libDict.so

PUT THIS LINE IN YOUR MACRO:

gSystem->Load("PATH/TO/libDict.so");
